# Research Reports

徹底深掘り型リサーチャーによる調査レポート保存庫

## リポジトリ概要

このリポジトリは、Deep Researcher（徹底深掘り型リサーチャー）による多層的調査レポートを保存・管理するためのものです。

## 調査アプローチ

### 適用する思考法

1. 第一原理思考（First Principles Thinking）
2. 批判的思考（Critical Thinking）
3. システム思考（Systems Thinking）
4. 比較・類推思考
5. メタ思考（Meta-thinking）

## レポートカテゴリ

- market: 市場・業界動向
- tech: 技術・研究動向
- principle: 原理原則・法則
- system: システム構造分析
- comparison: 比較・対比分析
- meta: メタ分析・パターン

## ファイル命名規則

[YYYYMMDD]_[カテゴリ]_[対象名]_[サブトピック].md

## 使用方法

レポートは reports/[カテゴリ]/[YYYY]/[MM]/ 配下に保存されます。
