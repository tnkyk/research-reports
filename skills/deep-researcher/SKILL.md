## レポート品質チェックリスト（必須）

レポート完成前に以下を確認：

### 基本構成チェック
- [ ] Executive Summary（1-2段落の要約）
- [ ] 各章が統一された構成（常識→発見者→発見の瞬間→検証→変化→日常生活）
- [ ] 結論と確信度の明示
- [ ] 参考文献リスト（一次史料と二次解釈の分離）

### 内容品質チェック
- [ ] **法則・原理の詳細説明**: 数学的表現、物理的意味、各記号の説明
- [ ] **具体例**: 数値計算、身近な例、驚きの事実
- [ ] **日常生活との関係**: 表形式で技術・現象・物理学の働きを対応
- [ ] **歴史解釈の多様性**: 複数の視点・解釈を提示（historyタイプ）
- [ ] **史料と解釈の分離**: 一次史料と二次解釈を明確に区別

### 数学的・物理的説明の要件

各法則・原理には以下を含める：

1. **数学的表現**
   - 公式の明示（LaTeX形式またはテキスト）
   - 各記号の定義と単位
   - 導出過程（必要に応じて）

2. **物理的意味**
   - 直感的な説明
   - 物理的イメージ
   - 例え話

3. **具体例**
   - 数値計算例
   - 身近な現象
   - 驚きの事実

4. **応用例**
   - 現代技術への応用
   - 日常生活での利用

### 読みやすさチェック
- [ ] 専門用語は初出時に説明
- [ ] 表を活用して視覚的に整理
- [ ] 段落は短め（3-4行程度）
- [ ] 重要な点は**太字**で強調
